## 🚀 خطوات التشغيل السريعة

### Step 1: التثبيت (1 دقيقة)
```bash
cd backend
npm install
```

### Step 2: الإعداد (2 دقيقة)
```bash
cp .env.example .env
# عدّل MONGODB_URI في .env
```

### Step 3: التشغيل (1 دقيقة)
```bash
npm start
# أو
npm run dev
```

### Step 4: الاختبار (2 دقيقة)
```bash
# في terminal جديد
curl http://localhost:3000/api/products
```

## 📞 الدعم السريع

### المشكلة: MongoDB connection failed
**الحل:** اقرأ [TESTING_GUIDE.md](./TESTING_GUIDE.md#-معالجة-المشاكل-الشائعة)

### المشكلة: Port 3000 already in use
**الحل:** غيّر PORT في .env

### المشكلة: Module not found mongoose
**الحل:** قشغل `npm install mongoose`

### المشكلة: لا أفهم الـ API
**الحل:** اقرأ [ORDER_SYSTEM_DOCUMENTATION.md](./ORDER_SYSTEM_DOCUMENTATION.md)
